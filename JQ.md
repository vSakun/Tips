	jQuery(document).ready(function($){

#### <a name="1">Селекторы по атрибутам</a>
- элементы, содержащие атрибут name - ```"[name]"```
- элементы, у которых значение атрибута name совподает с value - ```"[name=value]"```
- элементы, у которых значение атрибута name не совпадает с value - ```"[name !=value]"```
- элементы, у которых значение атрибута name начинается с value - ```"[name ^=value]"```
- элементы, у которых значение атрибута name заканчивается на value - ```"[name $=value]"```
- элементы, у которых значение атрибута name содержит подстроку value - ```"[name *=value]"```
- элементы, у которых значение атрибута name содержит слово value - ```"[name ~=value]"```
- элементы, у которых значение атрибута name имеют префикс value (равен value или имеет вид: "value-*") - ```"[name |=value]"```
- элементы, соответствующие всем заданным условиям на атрибуты одновременно - ```"[first][second][..."```
##### Пример: Соберёт все классы название которых НЕ "cont"
	jQuery(document).ready(function($){
	var j =  $("[class !=cont]");
	j.css({"border":"2px solid red"});
	});
#### <a name="2">Простые фильтры для выбора</a>
- первый и последний (соответственно) найденные элементы - ```":first" и ":last"```
- все найденные элементы, кроме указанных в selector - ```":not(selector)"```
- элементы с четными и нечётными(соответственно) номерами позиций, в наборе выбранных элементов - ```":even" и ":odd"```
- элементы с индексом превышающим N и меньшим, чем N(сщщтветственно) - ```":gt()" и ":lt()"```
- элементы, являющиеся заголовками(с тагами h1,..., h6) - ```":header"```
- элументы, которые в данный момент задействованы в анимации - ```":animated"```
##### Пример: Найдёт последний элемент среди элементов li
	jQuery(document).ready(function($){
	var j =  $("li:last");
	j.css({"border":"2px solid red"});
	});
#### <a name="3">Поиск по содержимому в элементах</a>
- элементы, содержащие заданный текст - ```":contains(text)"```
- элементы без содержимого(без текста и других элементов) - ```":empty"```
- элементы, которые содержат хотя бы один элемент из selector - ```":has(selector)"```
- непустые элементы - ```":perent"```
##### Пример: Найдёт span с текстом внутри "КОПИРАЙТ3"
	jQuery(document).ready(function($){
	$("span:contains(КОПИРАЙТ3)").css({"border":"2px solid green"});
	});
#### <a name="4">Поиск по дочерним элементам</a>
- элементы, расположенные первыми в своих родительских элементах - ```":first-child"```
- элементы, расположенные последними в своих родительских элементах - ```":last-child"```
- элементы, рассположенные определённым образом в родительских элементах(чётные, нечётные, идущие под хаданным номером) - ```":nth-child(значение)"```
- элементы, являющиеся единственными потомками в своих родительских элементах - ```":only-child"```
##### Пример: Найдём второй дочерний span в каждом родительском блоке
	jQuery(document).ready(function($){
	$("span:nth-child(2)").css({"border":"2px solid green"});
	});
##### Пример: Найдём каждый чётный span в родительском блоке
	jQuery(document).ready(function($){
	$("span:nth-child(even)").css({"border":"2px solid green"});
	});
##### Пример: Найдём каждый span находящийся на позиции 2n+3(каждый спад в бюлоке попадает в эти скобки и с каждым происходит это действие)
	jQuery(document).ready(function($){
	$("span:nth-child(2n+3)").css({"border":"2px solid green"});
	});
#### <a name="5">Поиск элементов в форме</a>
- элементы с тегом button или типом button - ```":button"```
- элементы, являющиеся переключателями - ```":radio"```
- элементы, являющиеся флажками - ```":checkbox"```
- элементы, являющиеся текстовыми полями - ```":text"```
- элементы, являющиеся полями ввода пороля - ```":password"```
- элементы, являющиеся полями загрузки файлов - ```":file"```
- элементы, являющиеся кнопками отправки формы - ```":submit"```
- элементы, являющиеся кнопками очистки формы - ```":reset"```
- элементы, являющиеся изображениями для отправки формы(input типа image) - ```":image"```
- элементы, являющиеся элементами формы(с тегами input, textarea или button) - ```":input"```
- выделенные элементы(со статусом selected). Это могут быть элементы типа <option> - ```":selected"```
- выделенные элементы(со статусом checked). Это могут быть элементы типа <checkbox> или <radio> - ```":checked"```
- активные элементы формы(со статусом enabled) - ```":enabled"```
- неактивные элементы формы(со статусом disabled) - ```":disabled"```
##### Пример: Найдёт элементы в ul на которых стоит статус cheked(выбран), например выбранные checkbox(border не отобразиться так как данное свойство не поддерживается radio и checkbox)
	jQuery(document).ready(function($){
	$("ul :checked").css({"border":"2px solid blue"});
	});
#### <a name="6">Использование нескольких селекторов</a>
- элементы удовлетворяющие любому из селекторов first, second, ... - ```"first, second, ..."```
- элементы из inner, которые являются потомками (т.е. лужат внутри) элементов из outer - ```"outer inner"```
- элементы из child, которые являются прямыми потомсками элементов из parent - ```"parent > child"```
- элементы из next, которые следует непосредственно за элементами из prev - ```"prev + next"```
- элементы из next, которые следует за элементами из parent - ```"prev ~ next"```
##### Пример: Выбрать все элементы span являющиеся братскими для p
	jQuery(document).ready(function($){
	$("p + span").css({"border":"2px solid blue"});
	});
#### <a name="7">Дополнительные селекторы</a>
- возвращает элемент в соответствии с номером индекса - ```":eq(n)"```
- возвращает первый братский элемент в выборке соответствующего типа - ```":first-of-type"```
- возвращает последний брасткий элемент в выборке соответствующего типа - ```":last-of-type"```
- возвращает элемент в соответствии с номером индекса(отсчёт ведётся с последнего элеменат выборки) - ```":nth-last-of-type(n)"```
- выбирает все дочерние элементы, с индексом n  в отношении братьев с тем же именем элемента - ```":nth-of-type(n)"```
- выберет все скрытые элементы - ```":hidden"```
- соответствует элементам со значение x атрибута lang - ```":lang(x)"```
- выбирает все дочерние элементы с индексом n, считая от последнего элемента - ```":nth-last-child(n)"```
- выбирает все элементы, у которых нет братьев, с тем же именем элемента - ```":only-of-type"```
- соответствует корневому элементу - ```":root"```
- соответствует всем видимым элементам - ```":visible"```
##### Пример: Выбрать все элементы li порядковый номер которых 4 с конца
	jQuery(document).ready(function($){
	$("li:nth-last-of-type(4)").css({"border":"2px solid blue"});
	});
#### <a name="8">Атрибуты и классы</a>
- возвращает/изменяет (в зависимости от числа параметров) значение атрибута - ```".attr("")"```
- удаляет атрибут - ```".removeAttr("")"```
- добавляет класс элементам на странице - ```".addClass("")"```
- удаляет класс(ы) у элементов на странице - ```".removeClass("")"```
- изменяет наличие классса у элементов на противоположное(добавляет/удаляет) - ```".toggleClass("")"```
- проверяет наличие класса cName хотя бы у одного из выбранных элементов - ```".hasClass("cName")"```
- возвращает/изменяет (в зависимости от числа параметров) значение атрибута value у элементов на странице - ```".val("")"```
- возвращает/изменяет значение свойств выбранных элементов - ```".prop("")"```
##### Пример: Добавлять/удалять класс prev по клики на заголовок
	jQuery(document).ready(function($){
	$("h1").toggleClass("prev");
	});
##### Пример: Проверяем есть ли класс prev у заголовка
	jQuery(document).ready(function($){
	var p = $("h1").hasClass("prev");
	alert(p);(вернёт true или folse)
	});
#### <a name="9">Работа со стилями</a>
- укажет растояние в пикселях слева до объекта - ```".offset().left"```
- укажет растояние в пикселях сверху до объекта - ```".offset().top"```
- сдвинет объект на указанные координаты в пикселях - ```".offset({top:100,left:300})"```
- установит всем элементам позицию (v_index - позиция элемента в выборке; v_value - текущее значение указанного метода)- ```".offset(function(v_index, v_value){ return {top:400,left:300}})"```
- укажет растояние слева от родительского элмента до указанного элемента(у родителя position должено быть указанно: absolute, relative, fixed) - ```".position().left"```
- укажет растояние сверху от родительского элмента до указанного элемента - ```".position().top"```
- возвращает ближайшего предка с позиционирование, отличным от static(позиционирование по умолчанию) - ```".offsetParent()"```
- возвращает/изменяет величину скроллинга (прокрутку) элемента - ```".scrollTop()"```
- озвращает/изменяет величину скроллинга (прокрутку) элемента - ```".scrollLeft()"```
#### <a name="10">Добавлние элементов (1 часть)</a>
- возвращает/изменяет(в зависимости от числа параметров) html-содержимое элементов на странице - ```".html()"```
- возвращает/изменяет(в зависимости от числа параметров) текст,находящийся в элементах на странице - ```".text()"```
- окружают элементы на странице заданными html-элементами - ```".wrap()"``` ```".wrapAll()"```
- окружают содержимое указанного блока заданными html-элементами - ```".wrapInner()"```
- удаляет родителя у выбранных элементов, а содержимое блока остаётся - ```".unwrap()"```
##### Пример: Удалим родителя у всех элементов li
	jQuery(document).ready(function($){
	$("li").unwrap();
	});
#### <a name="11">Добавлние элементов (2 часть)</a>
- добавлят заданное содержимое в конец элементов на странице - ```".append()"``` ```".appendTo()"```
- добавляет заданное содержимое в начало элементов на странице - ```".prepend()"``` ```".prependTo()"```
- добавляет заданное содержимоепосле элементов на странице - ```".after()"``` ```".insertAfter()"```
- добавляет заданное содержимое перед элементами на странице - ```".before()"``` ```".insertBefore()"```
##### Пример: Добавить в элемент с id #tut тег p
Используюя appendTo
	```jQuery(document).ready(function($){
	$("<p>NEW PUNKT</p>").appendTo($("#tut"));
	});```
Используя append
	```jQuery(document).ready(function($){
	$("#tut").append("<p>NEW PUNKT</p>");
	});```
#### <a name="12">Методы удаления элементов и содержимого</a>
- удаляет элемент - ```".remove()"```
- удаляет элемент (без потери связей объектов(обработчиков событий)) - ```".detach()"```
- удаляет cодержимое(сам элемент остаётся)(параметров не принимает) - ```".empty()"```
##### Пример: Удалить блок h1 и добавить его в конец блока #posts
	jQuery(document).ready(function($){
	var v = $("h1").detach();<!-- можно использовать и .remove() если нет назначенных на h1 оброботчиков событий -->
	v.appendTo("#posts");
	});
##### Пример: Удаляем только первые картинки
	jQuery(document).ready(function($){
	$("img").remove(":first");
	});
##### Пример: Удаляем только содержимое блока h1
	jQuery(document).ready(function($){
	$("h1").empty();<!-- в исходном коде страницы останется блок h1 -->
	});
#### <a name="13">Методы замены элементов</a>
- заменяет элемент(можно передовать ананимную функцию) - ```".replaceWith()"```
##### Пример: Заменить элемент h1 на текст content
	jQuery(document).ready(function($){
	$("h1").replaceWith("content");
	});
##### Пример: Заменить элемент h1 на тег p со значение content
	jQuery(document).ready(function($){
	$("h1").replaceWith("<p>content</p>");
	});
##### Пример: Заменить элемент h1 на все картинки со страницы
	jQuery(document).ready(function($){
	$("h1").replaceWith($("img"));
	});
- заменят элемент(применяется на выборку тех элементов которые будут вставлены вместо указанных в скобках(передоваемые как параметры)) - ```".replaceAll()"```
##### Пример: Заменить элемент h1 на все картинки со страницы
	jQuery(document).ready(function($){

#### <a name="14">Метод копирования элементов</a>
- копирует элементы - ```".clone()"```
##### Пример: скопировать элемент h1 и вставить в p
	jQuery(document).ready(function($){
	$("h1").clone().prependTo($("p"));<!-- варианты параметров .clone: 1.clone(folse)(при использовании clone без параметров foolse стоит по умолчанию). 2.clone(true) - клонирует с установленными обработчиками событий установленных на элемент. 3.clone(true, true) - копирует с установленными обработчиками событий в том числе указанных на дочернем элементе копируемого элемента. 4.clone(true, folse) - копирует с установленными обработчиками событий, а на дочерних элементах обработчики не будут работать -->
	});
##### Пример: скопировать, изменить и вставить элемент h1 в #header
	jQuery(document).ready(function($){
	$("h1").clone().text("новый текст").prependTo("#header");
	});
#### <a name="15">Перемещения по дереву DOM(после выборки)</a>
- Находит все дочерние элементы у выбранных элементов. При необходимости, можно указать селектор для фильтрации - ```".children()"```
- Находит ближайший, соответствующий заданному селектору элемент, из числа следующих: сам выбранный элемент, его родитель,его прародитель, и так далее, до начала дерева DOM - ```".closest()"```
- Находит элементы по заданному селектору, внутри выбранных элементов - ```".find()"```
- Находит элементы, которые лежат непосредственно после каждого из выбранных элементов - ```".next()"```
- Находит элементы, которые лежат после каждого из выбранных элементов - ```".nextAll()"```
- Находит элементы, которые лежат после каждого из выбранных элементов, но не дальше элемента, удовлетворяющего заданному селектору - ```".nextUntil()"```
- Возвращет ближайшего предка с позиционированием, отличных орт static (позиционирование по умолчанию) - ```".offsetParent()"```
- Находит родительские элементы у всех выбранных элементов - ```".parent()"```
- Находит всех предков у выбранных элементов, т.е. не только прямых родителй, но и прародителей и так далее, до начала дерева DOM - ```".parents()"```
- Находит предков, как и .parents(), но прекращает поиск перед элементом, удовлетворяющим заданному селектору - ```".parentsUntil()"```
- Находит элементы, которые лежат непосредственно перед каждым из выбранных элементов - ```".prev()"```
- Находит элементы, которые лежат перед каждым из выбранных элементов - ```".prevAll()"```
- Находит элементы, которые лежат перед каждым из выбранных, но не дальше элемента, соответствующего заданному селектору - ```".prevUntil()"```
- Находит все соседние элементы (под соседними понимаются элементы с общим родителем) - ```".siblings()"```
##### Пример: Выделить все теги "p" в элементе с классом ".cont"
	jQuery(document).ready(function($){
	$(".cont").children("p").css({"border":"1px solid red"});
	});
##### Пример: Найти в элементе с классом ".cont" последний элемент с тегом li
	jQuery(document).ready(function($){
	var res = $(".cont li:last");
	$(".cont").find(res).css({"border":"1px solid red"});
	});
##### Пример: Выделить следующий за первым элемент с тегом "li" в элепенте с классом ".cont"
	jQuery(document).ready(function($){
	var res = $(".cont li:last");
	$(".cont li:first").next().css({"border":"1px solid red"});
	});
#### <a name="16">Фильтрация выборки</a>
- Возвращает элемент, идущий под заданным номером в наборе выбранных элементов - ```".eq()"```
- Фильтрует набор выбранных элементов с помощью заданного селектора или функции - ```".filter()"```
- Возвращает первый элемент в наборке - ```".first()"```
- Фильтрует набор выбранных элементов, оставляя те, которые имеют потомков, соответсвующих селектору(возвращет либо true либо false) - ```".has()"```
- Проверяет, содержиться ли в наборе, хотя бы один элемент удовлетворяющий заданному селктору(возвращет либо true либо false) - ```".is()"```
- Возвращает последний элемент в выборке элементов - ```".last()"```
- Возвращает элементы, не соответствующие заданным условиям - ```".not()"```
- Возвращает элементы с индексами из определённой области(например от 0 до 5) - ```".slice()"```
##### Пример: Выбрать все элементы с классом ".cont" в которых есть элементы с тегом "li"
	jQuery(document).ready(function($){
	$(".cont").has("li").css({"border":"1px solid red"});
	});
##### Пример: Проверить, есть ли в элементе с классом ".cont" элементы с тегом "div", и если есть вывести сообщение "hello"
	jQuery(document).ready(function($){
	if($(".cont").is("div")){
	alert("hello");
	};
	});
##### Пример: Из выборки элементов "li" исключить первый элемент(v_index == 1)

#### <a name="25">Объект Event</a>
- методы
    * ```"event.isDefaultPrevented()"``` - проверяет вызывался ли метод event.preventDefault()
    * ```"event.isImmediatePropagationStopped()"``` - проверяет вызывался ли метод event.stopImmediatePropagation()
    * ```"event.isPropagationStopped()"``` - проверяет вызывался ли метод event.stopPropagation()
    * ```"event.preventDefault()"``` - отменяет действие по умолчанию
    * ```"event.stopPropagation()"``` - предотвращает пережачу события вверх от дочернего элемента к родительскому
    * ```"event.stopImmediatePropagation()"``` - останавливает выполнение дальнейших функций вызываемого события
- свойства
    * ```"event.currentTarget"``` - храниться элемент событие которого обрабатывается
    * ```"event.data"``` - храняться дополнительноо передоваемые переменные 
    * ```"event.delegateTarget"``` - элемент на котором установлен сработавший обработчик события
    * ```"event.pageX"``` - координаты курсора по оХ
    * ```"event.pageY"``` - координаты курсора по оУ
    * ```"event.namespace"``` - содержит строковое значение пространства имён произошедшего события
    * ```"event.relatedTarget"``` - элеент связанный с событием(Например при .mouseout() в данном свойстве будет объект элемента куда переместился курсор)
    * ```"event.result"``` - значение которое возвращает предыдущая фугкция обработчик вызываемого события
    * ```"event.target"``` - объект элемента который является источником события
    * ```"event.timeStamp"``` - содержит время когда произошло события с 1 января 1970 года (в секундах)
    * ```"event.type"``` - храниться тип произошедшего события
    * ```"event.which"``` - содержит информацию о нажатых клавишах
